The Apache Stanbol Talismane Integration

This Server provides in integration for Talismane with the Apache Stanbol Enhancer NLP module.

Talismane is used for Sentence detection, Tokenizing and Part of Speech tagging for French texts.

This Server supports two RESTful APIs:

parsed content
/analysis
Service that uses Talismane to analyse French texts and returns NLP processing results as JSON serialised AnalysedText content part.
/langidnet
NOTE: This server does NOT support language identification!
